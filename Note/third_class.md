## 本次课学的

- if-else
  - 与 Haskell 等 Functional Programming Language 不同的是
    JavaScript 中的 if-else 结构是不需要必须加上 eles 分支的
- switch
  - 注意，每一个statement都应该有一个break
  - 如果没有break就会“击穿”，相当于取并集
  - 一个switch应当有default情况
    (除非可以保证自己列举的情况是完全的)
  - default 一般不加 break (因为其一般在最后)
    当然，如果想加的话，也没有任何的关系
- 分支的嵌套
  - 从字面意思来说，也就是上面两种分支的嵌套，仅此而已
